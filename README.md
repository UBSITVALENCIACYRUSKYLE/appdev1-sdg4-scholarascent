# Scholar's Ascent — SDG 4: Quality Education

A Solo Leveling-inspired programming learning RPG where users gain EXP  and level up by completing programming lessons, quizzes, and challenges.

## SDG Chosen
SDG 4 — Quality Education

### Rationale
Access to quality education remains one of the most pressing global challenges. Scholar's Ascent addresses this by gamifying programming education, making it more engaging and accessible for learners of all levels. By combining RPG mechanics with real programming content, the app motivates users to keep learning through a rewarding progression system.
